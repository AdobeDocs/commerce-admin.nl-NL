# [!UICONTROL Sales] > [!UICONTROL Payment Methods] > [!UICONTROL PayPal Payflow Link]

>[!IMPORTANT]
>
>**PSD2 Eisen:** <br/>
>Vanaf 14 september 2019 zouden Europese banken betalingen kunnen terugdringen die niet voldoen aan [PSD 2](../../getting-started/compliance-payment-services-directive.md) eisen. Om te voldoen aan PSD2 [!DNL PayPal Payflow Link] moet worden geïntegreerd met [!DNL Cardinal Commerce]. Zie voor meer informatie [3-D veilig voor Payflow](https://developer.paypal.com/api/nvp-soap/payflow/3d-secure-overview/).
